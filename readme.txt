Gutenberg block to display post sharing options on socail media

== Description ==
Plugins lets you share your post on social media sites using gtenberg block. It is gutenberg block spin off of the social bar wordpress plugin 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/ctc-social-sharing` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Post screen look for CTC Social Sharing block on gutenberg section
4. Add the block
5. Select which social media to display from block setting
